# flac2ogg

convert flac music to ogg music

# Support Plaftform
    windows
    
# How to use
    $ps (run this command in Powershell)
       .\flac2ogg.ps1 -input_flac "La Luna - Sarah Brightman.flac" -output "La Luna - Sarah Brightman.ogg"
